# __name__ == __main__

如果程序运行在当前文件下，那么__name__ 就是main，如果是这个程序被作为别的程序的库，被import引用了，那么__name__ 就是模块名称

1、__name__这个系统变量显示了当前模块执行过程中的名称，如果当前程序运行在这个模块中，name 的名称就是__main__如果不是，则为这个模块的名称。

2、__main__一般作为函数的入口，类似于C语言，尤其在大型工程中，常常有if name == "main":来表明整个工程开始运行的入口。

# 集合运算（交并差集）

以下的集合运算，均会返回一个新的结果，不会改动原集合信息

使用 | 的操作符, 求并集（aset | bset 等同 aset.union(bset)）

使用 - 的操作符，求差集（aset - bset 等同 aset.difference(bset)）

使用 & 的操作符，求交集（aset & bset 等同 aset.intersection_update(bset)）

代码 解释
aset.union(bset) 并集 使用 union 函数，将两个集合进行合并并去重
aset.difference(bset) 差集（找出存在集合 A 但是不存在 集合 B 的元素）
aset.intersection(bset) 交集 使用 intersection 函数
aset.symmetric_difference(bset) 求不重合集 （计算两个集合中不重复的元素集合）
aset.isdisjoint(bset) 判断两集合是否有相同元素 bool
bset.issubset(aset) 判断是否是子集 （b是否是a的子集）bool

# 生成器

生成器（英文名 Generator ），是一个可以像迭代器那样使用for循环来获取元素的函数。

目前为止能想到的用途，就是封装循环体中用到的for i in range(...)循环，防止出错

虽然和列表推导氏形式很像，但别搞错了

mylist = [i for i in range(5)] # 这是列表推导式
gen = (i for i in range(5)) #这是生成器对象

# 生成器如何使用

  • 第一种方法，用for循环迭代出来
>>> gen = (x for x in range(3))
>>> for i in gen:
...     print(i)
...
0
1
2
  • 第二种方法，使用 next 方法一个一个地把元素取出来，如果元素全部取完了，生成器会抛出 StopIteration 的异常。（看不懂）
>>> gen = (x for x in range(3))
>>> gen
<generator object <genexpr> at 0x1072400b0>
>>> next(gen)
0
>>> next(gen)
1
>>> next(gen)
2
>>> next(gen)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
StopIteration

# 可变参数（NEW

最后还有一种非常特殊的参数，叫做可变参数。

意思是参数个数可变，可以是 0 个或者任意个，但是传参时不能指定参数名，通常使用 *args 和 **kw 来表示：

  • *args：接收到的所有按照位置参数方式传递进来的参数，是一个元组类型

  • **kw ：接收到的所有按照关键字参数方式传递进来的参数，是一个字典类型

    可变参数名 对应传参方法
    *args 位置参数传参 func(10, 20)
    **kw 关键字参数传参 func(a=10, c=30)
def func(*args, **kw):
    print(args)
    print(kw)

func(10, 20, c=20, d=40)

>>>(10, 20)
>>>{'c': 20, 'd': 40}

# 函数定义时的顺序问题

案例六：在定义时，必选参数一定要在可选参数的前面，不然运行时会报错

案例七：在定义时，可变位置参数一定要在可变关键字参数前面，不然运行时也会报错

## 必选参数一定要在可选参数的前面
def demo_func(a=1, b): # 错误的
...	SyntaxError: non-default argument follows default argument
def demo_func(a, b=1):  #正确的

## 可变位置参数一定要在可变关键字参数前面
def demo_func(**kw, *args): # 错误的
...	SyntaxError: invalid syntax
def demo_func(*args, **kw): #正确的

# 函数传参的大坑

函数参数传递的是实际对象的内存地址。如果参数是引用类型的数据类型（列表、字典等），在函数内部修改后，就算没有把修改后的值返回回去，外面的值其实也已经发生了变化。

>>> def add_item(item, source_list):
...     source_list.append(item)
...
>>> alist = [0,1]
>>> add_item(2, alist)
>>> alist
[0, 1, 2]

# 匿名函数搭配高阶函数（map，filter）

  • 需要注意如下几个点

    一：map 和 filter 函数返回的都不再是一个列表，而是一个迭代器对象。【用list()括起来就ok】

    二：reduce 不可以直接调用，而是要先导入才能使用👇

    from functools import reduce

  • map函数，可以将后面序列中的每一个元素做为参数传入lambda中。
>>> map(lambda x: x*2, [1,2,3,4,5])
[2, 4, 6, 8, 10]
  • filter函数，它会遍历后面序列中每一个元素，并将其做为参数传入lambda表达式中，当表达式返回 True，则元素会被保留下来，当表达式返回 False ，则元素会被丢弃。
>>>filter(lambda x: x < 0, range(-5, 5))
[-5, -4, -3, -2, -1]
  • reduce 函数，也是类似的。它的作用是先对序列中的第 1、2 个元素进行操作，得到的结果再与第三个数据用 lambda 函数运算，将其得到的结果再与第四个元素进行运算，以此类推下去直到后面没有元素了。
from functools import reduce
>>>reduce(lambda x,y: x+y, [1,2,3,4,5])
15
